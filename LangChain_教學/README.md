本課程專注於使用 LangChain 2.0 框架搭配 Ollama 本地模型，幫助您快速掌握如何在本地開發和部署 AI 應用。

## 課前準備

- 基本 Python 編程知識
- 安裝 Python 3.9+ 版本
- 具備命令行基本操作能力

## 課程目標

完成本課程後，您將能夠：
- 在本地部署和運行 AI 語言模型
- 使用 LangChain 2.0 構建複雜的 AI 應用
- 實現基於大型語言模型的各種實用案例
